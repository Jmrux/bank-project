# Proyecto de Banca
Proyecto para el curso de Programación Web. Simula la experiencia estandar de una aplicacion web de Banca.

## Descripción
Un proyecto creado inicialmente para un curso de Programación Web que buscaba enseñar sobre el uso de HTML, CSS, Javascript y el uso del Framework de React.
Posteriormente, se retomó el desarrollo de la aplicación para cumplir con las metricas de calidad impuestas por el analisis de codigo estatico de SONARQUBE, con el fin de aprender y entender los conceptos y usos de estas herramientas en el campo de la Calidad de Software.


## Tecnologias
- Desarrollado en su totalidad con Vite/React. 
- Metricas de calidad realizadas con SONARQUBE. 
- Pruebas unitarias y pruebas de componentes realizadas con Vitest y React Testing Library.
- Conexión con API implementada con Axios.
